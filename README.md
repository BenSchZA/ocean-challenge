# Ocean Challenge

## Initialize project

1. `nix-shell shell.nix`
2. `make init`

## Setup Environment

1. `nix-shell shell.nix`
2. `make ocean`
3. `make commons-ui`

## Contract Tests & Migration

1. `make test`
2. `make migrate`
